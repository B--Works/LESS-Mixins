# LESS Mixins

Some useful LESS mixins for common tasks

## Breakpoints

Quickly add media queries.

### Usage

```
.bp(768px, { display: block; });
```

## Browser Bars

Add [full width bars](http://css-tricks.com/full-browser-width-bars/) to an element. 

### Usage

```
.bars(color-value);
```

## Px to Em

Convert pixel values to em values.

### Usage

```
.em(max-width, 960px);
```
